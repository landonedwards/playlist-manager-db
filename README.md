# Playlist Manager DB

This application allows you to create, view, and upload playlists. 


## Instructions for Build and Use

Steps to build and/or run the software:

1. To use this website, click [here](https://playlist-manager-db.netlify.app/)

Instructions for using the software:

1. Create an account and click on the link sent to your email to log in (clicking on the email link is only necessary the first time). 
2. Create a playlist by titling it. To edit and add songs to it, click on the playlist link on the My Playlists page. 
3. View other people's playlists on the Explore page. If you like one, add it to your own playlists using the import to my account button. 

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Have the node package manager (npm) installed.
* Install the Supabase JS Client using npm. 
* Enter the command: npm create vite@latest to create a new Vite project.
